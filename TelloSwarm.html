Using a Hotspot

Step One:

Start by connecting to the individual drone with the computer being used. Then run the following python code to switch the drone to the alternate mode.

from dhitellopy import Tello
tello = Tello()
tello.connect()
tello.conntect_to_wifi('[Insert Wifi Name/SSID]', '[Insert Wifi Password}')
Step Two:

Connect the computer to the same wifi- has to be small like a hotspot.

Step Three:
Install Fing - Network Scanner.

This will allow you to scan for devices and find the ip adress used for each Tello.
