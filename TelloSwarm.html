Connecting Multiple Drones

Using a Hotspot

Step One:

Start by connecting to the individual drone with the computer being used. Then run the following python code to switch the drone to the alternate mode.

from dhitellopy import Tello
tello = Tello()
tello.connect()
tello.conntect_to_wifi('[Insert Wifi Name/SSID]', '[Insert Wifi Password}')
