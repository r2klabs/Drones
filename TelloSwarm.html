Drone Swarm Set-up



Connecting Multiple Drones

Using a Hotspot

Step One:

Start by connecting to the individual drone with the computer being used. Then run the following python code to switch the drone to the alternate mode.

from dhitellopy import Tello
tello = Tello()
tello.connect()
tello.conntect_to_wifi('[Insert Wifi Name/SSID]', '[Insert Wifi Password}')

After the drones LED goes dark, it should light up and cycle through all the colors before flashing yellow. At this point, you need to get the IP adress of the Tello. We used the app "Fling" to see IP adresses connected to the network, and turned the Tello on and off to see what IP adress it was. After you have the IP address, you can put it in a list on the TelloSwarm.py program.

from djitellopy import TelloSwarm

swarm = TelloSwarm.fromIps([

"192.168.53.192",
"192.168.53.124",
"192.168.53.85"

])

swarm.connect()
swarm.takeoff()

swarm.land()
swarm.end()
