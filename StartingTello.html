Getting the Drone Running

Uptdates:

Updates can be found through Tello app. (Not Tello EDU)

Updating Raspberry Pi:

 sudo apt update
sudo apt full-upgrade

Update Python:

Check current version-

python --version

Possible ways to update-

sudo apt update
sudo apt install python3.9

Collected From: Python Centeral, LinuxHint, RaspberryTips

Download DJITelloPy Libraries:

pip install djitellopy

Other options-

pip3 install djitellopy

Run Python Through Thonny

Switch Directories

Place Thonny code file on the desktop and look for the file location. Location can be found under properties. Should be something like "/home/piname/Desktop"

cd /home/piname/Desktop
python3 filename.py
